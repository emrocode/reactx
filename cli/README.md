## Features

- Rsbuild
- React
- React Router
- Tailwindcss
- ESLint and Prettier

## Usage

Use pnpm or any other package manager:

```bash
pnpm dlx setup-reactx@latest --dir my-app
```

```bash
cd ./my-app && pnpm run dev --open
```
